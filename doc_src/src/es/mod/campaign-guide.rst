Guía de creación de campañas
============================

Desde ``campaign.txt`` hasta los disparadores. Sintaxis completa: `Manual de creación de mapas <mapmaking.htm>`_.
.. contents::

----

Primeros pasos
--------------

Consulta la `Guía de mapas <map-guide.htm>`_ para un ``multi/*.txt`` de prueba primero, luego:

.. code-block:: text

   my_campaign/
     campaign.txt
     0.txt
     rules.txt

----
